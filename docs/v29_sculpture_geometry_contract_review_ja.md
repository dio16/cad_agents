# V29 Sculpture Geometry Contract Review

## 目的

feasible selected design を、次の full-geometry 実装へ渡す manifest に落とす。

## 分割

| Layer | Roles |
| --- | --- |
| slow macro | visible winding drum |
| medium transfer | barrel / transfer / riser / canted drive |
| fast fine | carrier / orbit / escape / pallet / balance |

## 出力の最低条件

- STEP / STL
- role manifest
- motion viewer
- visibility contact sheet
- printed / purchased BOM
- assembly sequence

## 禁止

- feasibility closure を飛ばして geometry へ入ること
- final object-value evidence を単なる色分けで代用すること
