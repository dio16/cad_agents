# V17 Layout-Locked Review

## 目的

V16は機構グラフ先行に変えたものの、CAD上では以下の問題が残っていました。

- 標準部品参照をAssemblyに入れていたため、宙に浮いたように見える部品があった。
- 脱進機表示モジュールが別部品で、ケージに実体支持されていなかった。
- ギア面の高さ分離が設計仕様として明確でなかった。

V17では、CAD製図前のレイアウト成立性を優先し、部品構成を整理しました。

## 変更内容

- Assembly投入部品を7個から5個へ削減。
- 標準部品はBOM/仕様に残し、Onshape Assemblyには投入しない。
- 脱進機表示モジュールを回転ケージ部品に統合。
- 駆動ギア面をZ18-23、固定リング/公転ピニオン面をZ27-34へ分離。
- 固定リング支持柱をベース側に保持し、リングが支持される配置にした。
- 公転ピニオンをケージ上の支持塔/軸穴で受ける設計にした。

## 改善できたこと

- 宙に浮く参照部品はなくなった。
- 固定リングと公転ピニオンの噛み合い平面が揃った。
- 手回しピニオンとケージ駆動ギアの平面が揃った。
- 表示モジュールがケージ一体になり、少なくとも支持されない別部品ではなくなった。

## まだ残る課題

- Onshape合致フィーチャーはまだ0。
- 手回し駆動ギアは下段に配置したため、鑑賞性としては視認しにくい。
- 本物のトゥールビヨン脱進機ではなく、ケージ回転と公転ピニオンを見せる機構オブジェクト段階。
- 3Dプリントで実動させるには、軸受座、止め輪、スペーサ、ネジ逃げ、組立順序をさらに設計する必要がある。
