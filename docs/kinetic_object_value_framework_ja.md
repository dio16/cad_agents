# キネティック・オブジェ価値フレームワーク（正本）

## 目的

この文書は、卓上トゥールビオンを「動く機構」から「置きたく、見続けたくなる機械彫刻」へ引き上げるための価値定義を定める。

## 価値命題

> 遠くから見ると大胆なキネティック・スカルプチャー、近づくと時計のように精緻で、通常は平面内で閉じる時計機構を立体空間へ解放した、自律給力型の卓上トゥールビオン機構オブジェ。

## 5 つの価値軸

| 軸 | 到達状態 | 失敗例 |
| --- | --- | --- |
| 遠景価値 | 輪郭、高低差、抜け、重心が一目で印象を残す | 円盤や土台が視線を占有し、機構が埋もれる |
| 中景価値 | 入力から出力までの力の流れを 1 hero view で追える | 動いてはいるが、何が何を動かすか分からない |
| 近景価値 | 高速・微小部ほど薄く、緻密で、時計らしい密度がある | すべてが同じ太さ・粗さで、細部に報酬がない |
| 時間価値 | 大・中・小の異なるテンポが重なり、周期内に見せ場がある | 全体が同じ速度で回るだけ |
| 空間価値 | 異なる高さまたは軸方向をまたぐ駆動が必然として見える | 平面時計を厚くしただけ |

## 受入条件

完成 claim には、次を同一世代で示す。

1. **value brief**
   - 想定鑑賞距離、置かれる場所、第一印象、近接時の発見、避けるべき印象。
2. **hero-view set**
   - 遠景、斜視、近景、立体駆動の 4 視点。
3. **motion legibility review**
   - `autonomous winding -> energy transfer -> carrier -> orbit -> escapement -> balance` が 1 枚または 1 クリップで説明できる。
4. **layered-rhythm review**
   - 遅い大運動、中速の伝達、速い微小運動がそれぞれ視認できる。
5. **spatial-drive review**
   - 高さまたは軸方向をまたぐ伝達が、視覚 gimmick ではなく機構上の必然として存在する。
6. **silhouette / openness review**
   - 見せ場を隠す主構造がなく、視線が自然に入力から細部へ降りる。

## 判定語

- `mechanism prototype`: 機構成立のみ。
- `kinetic concept`: 価値 brief はあるが、実装または証拠が未完。
- `spatial-kinetic-sculpture-object`: 5 軸が証拠付きで成立し、機械・配布物 gate も pass。

## 禁止する短絡

1. 視認できるだけで「美しい」とみなさない。
2. 透明化や色分けだけで「動きが分かる」とみなさない。
3. 立体配置だけで「立体駆動」とみなさない。力の伝達が立体である必要がある。
4. 自律給力の見た目だけで claim しない。低電圧モータ、巻上げ列、蓄力、出力への伝達契約が必要である。
