# V29 以降の空間キネティック・スカルプチャー化ワークプラン

## 目的

V28 を「機構成立の基準点」とし、その上に次の新ゴールを実装する。

> 大胆なキネティック・スカルプチャー全体像、時計のような精緻な高速部、自律給力入力、立体空間を使う駆動。

## フェーズ

### Phase 0 — Goal reset

- V28 を旧ゴール達成の baseline に降格
- 新しい completion claim / goal-state / object-value gate を正本へ反映

### Phase 1 — Value brief

- 遠景・中景・近景で何を見せるか
- 自律給力入力の所作
- 立体駆動の驚き
- 避けるべき見た目

### Phase 2 — Concept options

少なくとも 3 案を比較する。

1. 垂直軸 rotor + 水平 tourbillon
2. 傾斜 carrier + 斜交軸伝達
3. 多層ブリッジ + 立体歯車列

比較軸は、価値一致、機構成立、製造性、検証負荷、将来拡張性。

### Phase 3 — Pre-CAD feasibility package

- 自律給力 architecture
- spatial-drive map
- transmission graph
- gear / shaft / support first-pass dimensions
- view hierarchy
- failure pre-mortem
- reject reasons for non-selected concepts

### Phase 4 — Selected concept contract

- manifest schema
- role map
- standard hardware
- gate matrix
- object-value evidence plan

### Phase 5 — Implementation

- まず最小の architecture demonstrator
- 次に full sculpture geometry
- 最後に fabrication package

### Phase 6 — Confirmation

- mechanical truth
- autonomous-winding truth
- spatial-drive truth
- object-value review
- deliverable package

## Exit conditions

次を満たすまで完成と呼ばない。

1. `spatial-kinetic-sculpture-object` claim
2. 自律給力入力の物理契約
3. 立体駆動の物理契約
4. 5 軸の object-value review pass
5. fabrication / deliverable / README handoff pass
